# 環境光テルミン

　環境光センサーを用いた楽器アプリです。

　以下の機能があります。

* 環境光センサーの値に応じて音程を変化
* 最後のタッチ位置に応じて音量を変化
* 音程と音量に応じて画面表示を変化
* 接近を検知して音を止める
* アプリ起動から10秒間周辺の明るさを測定して音程を調整

## 使い方

* 端末をホームボタンが左にくる方向に置きます
* 10秒間待ちます
* 画面をタッチして音量を調整しながら環境光センサーに手をかざしたり端末の角度を変えたりして音程を調整して演奏します
  * 左をタッチするほど音量が大きくなります
  * 右をタッチするほど音量が小さくなります
  * 暗くなると音程が下がります
  * 明るくなると音程が上がります
